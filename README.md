# 📚 BukuKu - Aplikasi Katalog Buku Android

Proyek ini dibuat untuk memenuhi tugas mata kuliah dengan mengimplementasikan aplikasi katalog buku berbasis Android (Jetpack Compose) yang terintegrasi dengan REST API (PHP & MySQL).

## ✨ Fitur Utama
* **Google Sign-In:** Otentikasi aman menggunakan Google Credential Manager.
* **Katalog Komunitas:** Menampilkan semua buku yang diunggah oleh seluruh pengguna.
* **Fitur Tab "Buku Saya":** Memfilter daftar agar pengguna hanya melihat buku buatan mereka sendiri.
* **Manajemen CRUD:** Tambah buku (dengan fitur *crop* gambar), edit detail, dan hapus buku milik sendiri.
* **Sistem Favorit:** Menandai buku yang disukai dengan ikon hati.

---

## 🛠️ Teknologi yang Digunakan
* **Frontend:** Android Studio, Jetpack Compose, MVVM Architecture, Retrofit 2, Coil (Image Loader).
* **Backend & Database:** PHP native (REST API) & MySQL.
* **Hosting Server:** InfinityFree (Free Hosting).
